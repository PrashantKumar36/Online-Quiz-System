# Online Quiz System

A web-based quiz application built using Python Flask.

---

## Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript
- SQLite

---

## Features

### Student Features

- Register/Login
- Attempt Quizzes
- Timer System
- Instant Results
- View Correct Answers

### Admin Features

- Admin Login
- Create Quiz Categories
- Add Questions
- Delete Questions
- View Student Scores

---

## How To Run

### Install Libraries

pip install flask flask_sqlalchemy

### Run Project

python app.py

### Open Browser

http://127.0.0.1:5000
